Doomsday Multiplatform Enhanced Doom Source Port


The Doomsday Engine is a greatly enhanced DOOM source port available for Windows, Mac OS X, Linux and most other Unix platforms (source code is provided).

Doomsday supports many games including the classic first-person shooters DOOM, Heretic and Hexen.

They aim to retain the original feel of the games yet leverage modern techniques and technologies, including hardware accelerated 3D graphics and client/server networking. A lot of emphasis is placed on good-looking graphics.
