Adobe Photoshop CC 14.0 Portable | 486.6 MB

Adobe Photoshop CC - new, the next version of CS6 software for processing raster graphics. Supports numerous graphic formats. Adobe Photoshop allows you to create new images and edit them. Photoshop used to create photorealistic images to work with color scanned images, retouching, color correction, transformation of graphs, color separation, etc.Adobe Photoshop has all the methods of working with bitmaps, in this case is to work with layers and uses the contours. The program is the undisputed leader among professional graphic editors due to its widest possibilities, high efficiency and speed. Adobe Photoshop provides all the necessary tools for correcting, editing, preparing images for printing and high quality output.

New features in Adobe Photoshop CC:
• The new "Smart Sharpen". A variety of textures, sharp contours and clear details. The new "Smart Sharpen" is the most modern technology to increase the sharpness of today. It analyzes the image, allowing you to maximize clarity, and significantly reduce the noise and ghosting to create high-quality images that look natural.
• Smart increasing sampling. Increase the low resolution image for use in print or image to the size of the average quality of the billboard or poster. A new feature allows you to keep up-sampling detail and sharpness without making noise.
• Additional functions. As a component of Creative Cloud, Photoshop app gives you all the power of image editing and video editing, and also includes advanced tools for editing three-dimensional objects and analysis of images, previously only available in Photoshop Extended.
• Camera Raw 8 and layer support. Apply edits Camera Raw as a filter for layers and files in Photoshop, and then Upgrade layers and files you fit properly. In addition, with the new external module Adobe ® Camera Raw 8, you can more accurately correct the images, eliminate perspective distortion and create a vignette.
• Variable rectangles with rounded corners. This is one of the most requested features. Now you can change the size of the figures, as well as edit them, both before and after its creation. You can even change the radius of the corners of rectangles. If the figure is intended to be placed on a web page, you can export data from the CSS file in order to save time.
• Ability to simultaneously select multiple contours and shapes. Increase the efficiency of their business processes, quickly picking contours, shapes and vector masks in just a few clicks of the mouse. Even in a multi-document with a lot of loops can easily find the right path (and layer) directly on the canvas, using the new filter mode.
• Image stabilization chamber. Take photos even a moving camera. If the picture is blurry due to the slow shutter speed or a large focal length, the function "Image Stabilization Camera" analyzes the trajectory and restores sharpness.
• Expanded support for smart objects. Apply effects, picture blur and plastic materials with the possibility of reversible correction with support for smart objects. Add blur effects, pushing, pulling, wrinkling and swelling pictures or video, while maintaining the ability to go back to the original file. Change or remove the effects whenever you want, even after saving the file.
• Improved drawing of three-dimensional objects. When working with three-dimensional objects and textures to create interactive maps of the preview can now be up to 100 times faster. Thanks to the powerful mechanism of drawing in Photoshop, you can create stunning three-dimensional model.
• Improved font styles. You spend hours on the whole design of the font. The use of font styles will keep the formatting as a style, and then apply it with one click. You can even specify font styles to be used in all your documents Photoshop.
• Copy the attributes of CSS. When creating a web design code manually, you can always create elements such as rounded corners or color corresponding to the original. Photoshop can now create CSS code for the specific design elements. You only need to copy and paste the code into your web editor to get the desired result.
• Conditional actions. Automate routine operations using conventional machining operations. To automatically select the different actions based on the rules you create in these commands are used instructions if / then.
• Improved panel «3D-scene." With the ease of changing from two-dimensional to three-dimensional editing thanks to the panel «3D-scene", which contains a set of parameters familiar to you, with whom you have worked before in the "Layers", such as "Duplicate", "Create an instance", "Groups" and "Delete".
• The functions that help you save time. Reduce time spent on routine tasks, thanks to the small, but very user-friendly features, users are asked to realize that. Quickly create contours with new keyboard shortcuts, mouse, move the contours of space, include ICC profiles in formats PNG, and more.
• Improved 3D-effects. Shadows and reflections can turn your 3D-illustration of a masterpiece, and interactive preview images of higher quality will create memorable works even faster. In addition, you can now easily create glow effects, stage lighting, as well as highlighting the reliefs and textures of higher quality.
• Import colors from the web files. Import the color samples directly from files HTML, CSS and SVG. Change these samples as you wish or just use the ready-made color schemes are based on an existing web content.
• System font smoothing. With the parameter, which smoothes the corresponding smoothing that is implemented on the Mac or Windows ®, you can now create realistic previews that allow you to see how a font will look on your web pages.
• Technology based editing of content. Retouch images with tremendous precision and ease. Select the object image that you want to delete, increase, decrease, or move. Editing technology, taking into account the contents of the Fill, insert patch or recomposition image.
• The mechanism of Mercury Graphics Engine. Click and drag an object - changes are immediately displayed on the screen, even when working with large image files. The mechanism of Adobe Mercury Graphics Engine provides high-speed operation of key tools such as "plastic" and "Puppet Warp," which makes it easy to edit pictures and get immediate results.
• Powerful design tools. Create a Photoshop design for any media, using a versatile set of tools. Use font styles, editable shapes, vector layers that allow you to apply strokes and gradients to vector objects, tools for quickly creating strokes and dashed lines, and many other features.
• Intuitive video creation. Use editing skills in Photoshop environment when performing the editing. Upgrade clips, using all the usual tools Photoshop, and create movies with a simple-to-use set of tools for working with video. If you need to perform more complex operations installation, refer to the software Adobe Premiere ® Pro.
• Gallery blur. Add photographic blur effects using simple controls directly on the canvas. Create the effects of tilt and shift, follow the blurring of the image and increase the sharpness of a single point of focus, and use different types of blur to a certain point. Through the mechanism of Mercury Graphics Engine you can quickly see the effect applied, and support for Smart Objects can be removed if necessary effect and return to the original file.
• Intuitive tools and support for DICOM. Take advantage of advanced features for editing of 3D-objects and medical imaging. Edit the 3D-objects using the user-friendly interface with contextual controls on the canvas, and easily analyze and use the files DICOM.
• Background saving and auto-recovery. Now you can continue to operate even when the background storing large files Photoshop. Use the auto-recovery function to save the changes while staying focused on the progress made.
• Improved filters "Minimum" and "Maximum". Create more accurate masks and selections, using filters "Minimum" and "Maximum", which now include an option for saving a square or round objects.

Among the most interesting innovations are the following:
• «Smart" sharpening algorithms reduce the halo effect and the noise gain at higher mikrokontrasta.
• Smart upsampling - a powerful and versatile tool for increasing the size of the image, whether it's a picture with the phone or quality photograph that you want to "stretch" to poster size.
• Camera Raw 8 - is used as a filter to a layer or file, including video files, includes the effects of gradients, automatic perspective correction, elimination of defects is shown in the beta version of Adobe Lightroom 5.
• Camera Shake Reduction - the long-awaited mechanism capable of analyzing the image to determine the "path" of blurring caused by the displacement of the camera or the photographer hand tremor, and "restore" the sharpness of the image (see video below).
• Advanced functionality of Smart Object. The objects are now supported by non-destructive, parametric filters, overlay family Blur.
• Conditional action - incredibly useful in batch processing images scripts Actions received a conditional "if, then." Allows you to automatically run a variety of operating procedures for the implementation of certain conditions.

System requirements:
Intel Pentium 4 or AMD Athlon 64 (2 GHz or faster)
Microsoft Windows 7 or Windows 8
1 GB of RAM
The screen is 1024 x 768 (1280 x 800) with 16-bit video card and 512 MB of video memory (support for 3D-components requires 1 GB of video memory)
System with support for OpenGL 2.0

Note:
Select interface language to run the portable building will be in the file ChooseLang.ini near PhotoshopCCPortable.exe
Accelerate launch! In the folder ".. PhotoshopAppPsResources" are some of the resources of the program that you will be able to manipulate (ie, disconnect and connect them as needed). There is the ability to edit video files. To turn off the video editing (if you do not use it and want to accelerate the launch of the program) - dynamiclink delete folders and folders of dynamiclinkmediaserver ".. PhotoshopAppPsResources" or (if you want to turn off only for a while), rename it to "-dynamiclink" and "-dynamiclinkmediaserver" and to enable the reverse steps.
Up module Camera Raw 8.0.0.137, but no lens profiles. To add profiles lenses Camera Raw and Lens Correction unzip ".. PhotoshopAppPsResourcesCameraRaw" (originally a portable building is just a blank template). If you want to be for some time to disable Camera Profiles and Lens Profiles (to the same and run the program to accelerate), then remove them is not necessary: ​​just place at the beginning of the folder name CameraRaw "-" sign or the other (example: "-CameraRaw "), and to enable the reverse steps.
And finally, a word of advice: when you close it, especially if you are working from a flash drive, wait a few seconds before restarting (or before removing the USB-device), as the program needs some time to move from a system and registry in your folder the updates and remove the traces
