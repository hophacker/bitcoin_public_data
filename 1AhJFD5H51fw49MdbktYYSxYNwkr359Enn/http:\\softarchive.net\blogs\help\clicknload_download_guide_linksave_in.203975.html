Click'N'Load Download Guide (Linksave.In)

Help Center
Click'N'Load is the built in jDownloader's container that does not require the downloading on the container file to your computer/desktop when you wish to download the files using jDownloader.

It is required to install jDownloader and Click'N'Load before you can start to download images from my threads if We have used Linksave.in's Click'NLoad protection services to protect the downloads We made.

Oops! Something is hidden :)
Looks like you're unregistered user :( Log in, or take a few seconds to register new account ;)

Install and let's it automatic update latest plugin.


Step 1. Install jDownloader and Click'N'Load Plugin



Step 2. Click on the Click'N'Load button on the Linksave.In's page



Step 3. Wait for the container to be automatically loaded in jDownloader



Step 4. It will automatically check links online statuses

Step 5. Click "Continue" to start your download immediately!


You can easy add your premium account to download with JDownloader

FAQ :
- Why not using CryptLoad ? CryptLoad using CCF, DDF ... containner. And They gets decrypted easier , links get reported. Until nơw not with Click'n'Load.
- Why using JDownloader ? JDownloader supported Windows, Mac, Linux ... than Cryptload only can deal with Windows


*******************************************************
Troubleshooting:
What to do when an error says "the dlc container is invalid"?

Pretty simple, just wait for 30 minutes or the next day and try again. Sometime, LinkSave.In's servers are busy that causes the CNL resolving issue , no big deal actually.
Just try again in about 30 - 60 minutes later, or the following day, it should work.

Post any other issues if you have under this article or Pm me , We will try to support your problems the best We can.
