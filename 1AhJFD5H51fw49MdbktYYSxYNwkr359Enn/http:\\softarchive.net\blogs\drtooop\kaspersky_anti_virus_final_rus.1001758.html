~ The main components of protection:
• File Anti-Virus
File Anti-Virus prevents infection of computer's file system. The component starts when the operating system resides in RAM, and scans all opened, saved and run files on your computer and all attached disk drives. Kaspersky Anti-Virus intercepts each access to the file and checks the file for known viruses. Further work on the file is only possible if the file is not infected or is successfully treated by the program. If the file for some reason can not be cured, it will be removed. A copy of the file will be saved in the backup, or placed in quarantine.

• Mail Anti-Virus
Mail Anti-Virus scans incoming and outgoing mail messages on your computer. A letter will be available to the addressee only, if it does not contain any hazardous objects.

• Web Anti-Virus
Web-Kaspersky Anti-Virus intercepts and blocks scripts on a web site if it is a threat. Strict monitoring all HTTP-traffic. In addition, the component blocks access to dangerous Web sites.

• IM-Anti
IM-Virus provides security while working with instant messengers. The component protects information that comes to your computer protocol instant messengers. IM-Virus provides safe operation of various programs designed for instant messaging.

• Proactive protection
Proactive protection can detect a new malicious program before it has time to cause damage. The work component is based on monitoring and analyzing the behavior of all programs installed on your computer. Depending on the actions they perform Kaspersky Anti-Virus makes a decision on whether each program is potentially dangerous. Thus, your PC is protected not only from known viruses but from new, not yet investigated.

• Anti-Phishing
Component built into the Web Anti-Virus and IM-Virus, which allows you to check the web address on the membership lists of suspected phishing and web addresses.
