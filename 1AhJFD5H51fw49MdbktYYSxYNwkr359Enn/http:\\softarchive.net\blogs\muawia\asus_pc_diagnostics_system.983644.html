Asus Pc Diagnostics system this is a handy utility for those who have a Asus motherboard for PC (Laptop and Desktop) and want to test their properties panel, and even if you do not encounter a problem in the work of the motherboard, well is to know that everything is working properly! This tool allows you to fully test the components of computers, keyboards on laptops, memory, CPU, video card, sound card, etc. .... The System Information Collection detects the essential operating system and devices information.

The System Devices Test detects and reports any errors on the physical devices such as the current system status and performance.

The System Stress Test checks the system's stability and reports any errors on the main components such as the CPU, memory, and display adapter.

NOTE:.
ASUS PC Diagnostics will work only with Asus hardware.
