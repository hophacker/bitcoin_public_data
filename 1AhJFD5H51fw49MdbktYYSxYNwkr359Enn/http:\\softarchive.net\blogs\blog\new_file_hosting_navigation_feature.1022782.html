New File Hosting Navigation Feature

SA News


We want to present you our new FileHostings Feature
Yes...........Since Long we have not give any update to our users, but time has come to make and let you know about it!

In this Speedy Word, FileHostings are changing depending on different hard conditions, so keep this in mind, we are introducing new file hosintg Naviagation on our site for our dear users.

Now all articles will have FileHostings icons which you can see with the title of the post in different colors.



If you click on any icon of FileHosting you will be on page where only news with specific filehosting will be found.
For example: you need only news which have RapidGator File hosting.
Click to RG icon and whola http://filehosting.softarchive.net/rg/.

And thats not all!

Go to your  "MY PROFILE" And You will found
"File hosting service filter"
Yes! Waiting ends and you get what you were looking for.
Default - "Show all news".



Want to see only news with RapidGator? ok
uncheck  "Show all news" => check "RapidGator" => Save Changes
And now you will see news which have must RapidGator Filehosting !
Multiple selected also allowed, you can check 1-2-3-4-...etc filehosts which you want.

Hope you will like this feature!
Always yours SA Team
